FileStoreBot - Telegram File Store Bot (Pyrogram)
-------------------------------------------------
1. Open config.py
   - Replace YOUR_BOT_TOKEN_HERE with your bot token
   - Replace YOUR_MONGODB_URI_HERE with your MongoDB URI

2. Upload to GitHub and connect to Render or Railway.

3. Dependencies are listed in requirements.txt.

4. Procfile ensures Render runs: python3 main.py
